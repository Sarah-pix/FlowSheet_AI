# FlowSheet_AI
 AI-Powered Excel Query Automation System


## Overview
FlowSheet_AI is a smart automation workflow built with n8n that helps analyze supplier data directly from Excel files using AI.

Instead of manually checking rows, comparing vendors, and creating reports, this workflow automates the entire process. It reads supplier data from an Excel sheet, sends the information to an AI procurement analyst, processes the responses, and generates a clean final report automatically.

The goal of this project is to reduce manual effort, save time, and make supplier analysis faster and more efficient.

## Features
- Read supplier data from Excel sheets
- Automatically process selected rows
- AI-based supplier and procurement analysis
- Structured response parsing
- Aggregate all AI-generated insights
- Generate downloadable Excel reports
- Fully automated workflow using n8n

## How the Workflow Works
1. User uploads supplier Excel data
2. The workflow reads and filters the rows
3. Each supplier record is processed one by one
4. AI analyzes the supplier information
5. Responses are parsed and organized
6. Final results are combined
7. A new Excel analysis report is generated automatically
